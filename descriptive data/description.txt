งานนี้ประกอบด้วย modules หลักสามตัวคือ
1. model
    ใช้จำลองฐานข้อมูลที่จำเป็นคือ Cow และ Team
    class Cow นั้นใช้เก็บทั้งข้อมูลของวัวภายในโปรแกรม และเก็บ method ที่เกี่ยวข้องที่ช่วยทำงาน เช่น set strike ซึ่งจะทำงานในแต่ละรอบการเล่น
    class Team ใช้รวมทีม วัวทั้งสามตัว และมี method ช่วยเหลือ เพื่อนับคะแนนทีม

2. view
    class GameView เป็นคลาสหลักสำหรับ วาง GUI ของโปรแกรม 
    โดยจะมี components หลักคือ 
    1. gameLog เอาไว้แสดงว่า วัวแต่ละตัวได้คะแนนเท่าไหร่
    2. ปุ่ม start game ใช้เริ่มจำลองการเล่นเกมของวัวใหม่
    3. rankingText ส่วนแสดงอันดับ คะแนนของวัวแต่ละตัว 
    4. teamWinnerLabel ส่วนแสดงรายงานผู้ชนะ

3. controller
    class BowlingController ใช้จัดการการนับคะแนน และการเริ่มเล่นเกม ที่คลาสนี้
